# Python依赖安装

## 1. 通过pip安装
每章或者小的案例会配置相应的requirements.txt
通过命令安装

pip install -r requirements.txt

### 参考链接
（1）最新pip下载地址：https://pypi.python.org/pypi/pip
（2）Anaconda自带pip

## 2. 通过.whl文件进行安装
```cmd
pip install ×××.whl
```
### 参考下载链接
